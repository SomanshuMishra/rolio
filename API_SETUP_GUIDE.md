## **Option 2: Google Gemini API (Recommended for Real Job Matching)** ⭐

**Best free option - unlimited job searches, no credit card needed.**

### Key Details:
- **Cost**: Completely free (60 requests/minute)
- **Speed**: Fast responses
- **Credit Card**: NOT required
- **Setup Time**: 5 minutes

### Step-by-Step Setup:

#### Step 1: Get Your Free API Key
1. Open your browser and go to: https://aistudio.google.com/app/apikey
2. Click the **"Get API Key"** button
3. Click **"Create API key in new project"**
4. Copy the new API key (it starts with "AIza...")
5. Keep it safe (don't share it)

#### Step 2: Add to Auto Apply Jobs
1. Open Auto Apply Jobs
2. Go to **Dashboard → Settings**
3. Find the **"AI Provider"** section
4. Make sure **"Google Gemini - Free & Recommended"** is selected
5. Paste your API key into the **"API Key"** field
6. Click **"Save API Key"** button
7. You'll see a green success message ✓

#### Step 3: Start Using
1. You can now upload your resume and search jobs
2. The app will use Google Gemini to find best matches
3. No more limits - search as many jobs as you want!

## **Option 3: Groq API (Free & Super Fast Alternative)** ⚡

**Alternative free option - if you want ultra-fast responses.**

### Key Details:
- **Cost**: Completely free (30 requests/minute)
- **Speed**: Extremely fast (better UX)
- **Credit Card**: NOT required
- **Setup Time**: 5 minutes

### Step-by-Step Setup:

#### Step 1: Create Groq Account
1. Go to: https://console.groq.com
2. Sign up with your email (completely free)
3. Verify your email
4. Log in to the dashboard

#### Step 2: Get Your API Key
1. Look for **"API Keys"** in the navigation menu
2. Click **"Create New Key"**
3. Copy your new API key
4. Keep it safe

#### Step 3: Add to Auto Apply Jobs
1. Open Auto Apply Jobs
2. Go to **Dashboard → Settings**
3. Find the **"AI Provider"** dropdown
4. Select **"Groq - Free & Super Fast"**
5. Paste your API key into the **"API Key"** field
6. Click **"Save API Key"** button
7. Done! Enjoy the speed 🚀

## **Troubleshooting**

### "Invalid API Key" Error
- Make sure you copied the **entire key** (no spaces before/after)
- Check that the key came from the correct service
- Try the alternative provider (Groq instead of Google)

### "Rate Limited" Message
- Free tiers have request limits (60/min for Google, 30/min for Groq)
- Wait a few minutes and try again
- Use Demo Mode while waiting

### "No Results" from Jobs Search
- Make sure your resume is uploaded successfully
- Check that your job preferences aren't too restrictive (salary range, location)
- Try Demo Mode to verify the app is working
